联系人：aaaa@aaa.com

## 接口域名

```
http://www.zfw.com/api/v1
```

## 用户登录

### uri

`/login`

### 请求方式

`post`

### 请求参数

> post请求体中传递数据

| 字段名   | 类型   | 说明 |
| -------- | ------ | ---- |
| username | string | 账号 |
| password | string | 密码 |

### 返回值

`json`

| 字段名 | 类型   | 说明           |
| ------ | ------ | -------------- |
| expire | int    | 有效期 单位 秒 |
| token  | string | 登录的token值  |

```json
{
    "expire": 7200,
    "token": "eyJ0eXAiOiJKV"
}
```

### 错误码

| 错误码 | 说明                 |
| ------ | -------------------- |
| 1000   | 用户名或密码不正确   |
| 1001   | 用户名或密码没有传递 |
